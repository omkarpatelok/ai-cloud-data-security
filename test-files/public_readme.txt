CloudScanner Project - README
==============================

CloudScanner is an open-source cloud security scanning tool.
It helps organizations monitor and protect their cloud infrastructure.

Features:
- Automated S3 bucket scanning
- Data classification and tagging
- Risk scoring and compliance reporting
- Real-time alerting for misconfigurations

Getting Started:
1. Clone the repository
2. Configure your AWS credentials
3. Run the application
4. Access the dashboard at http://localhost:8081
